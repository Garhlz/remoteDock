## 推荐路线

配置持久化已经完成。当前最值得继续做的是“主机管理”：在界面里新增、修改和删除主机，而不是手动编辑 JSON。

建议顺序：

1. 本地 JSON 配置

3. VS Code Remote / Tailscale 等常用动作

5. 菜单栏、图标、视觉 polish
